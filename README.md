# 指尖非遗：景德镇制瓷（高保真版）

基于 Three.js 的交互式景德镇制瓷体验网页：选泥 → 定型 → 拉坯 → 利坯 → 施釉 → 烧制。

## 本地运行

用任意静态服务器打开根目录即可（需支持 ES Module），例如：

```bash
npx serve
# 或
python -m http.server 8000
```

然后访问 http://localhost:3000 或 http://localhost:8000

## 部署为网站（GitHub Pages）

详细步骤见 **[DEPLOY.md](DEPLOY.md)**：包含本地运行、上传 GitHub、开启 Pages 的完整说明。  
开启后网站地址：`https://<你的用户名>.github.io/<仓库名>/`
